Welcome to the help page !





To create a new simulation, by clicking the "Create Simulation" button you can access the simulation page. After this step toolbox will be displayed.




Toolbox consist of queue, connection and server components. You need to drag and drop each component one by one. There should be a connection component between queue and server,otherwise simulation will not run because of the missing connection error.




After completing the drag and drop steps you need to click on Run Simulation button from the navbar.



It redirects to another page that asks about the parameters. You need to provide each value otherwise the simulation will not work because of missing parameter error.
