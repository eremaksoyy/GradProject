To create a new simulation, by clicking the "Create Simulation" button, the simulation type selection

page can be accessed. After this step you can choose the simulation type.






The selection page displays four types of queuing simulations: M/M/1, M/M/1/L, M/M/C, and M/M/C/L.

Simulation type can be selected by clicking on it. After this step the toolbox will be displayed.






Toolbox consist of queue, connection and server components. Each component should be dragged

and dropped one by one. At least one server and queue component should be dropped on the

canvas to run the simulation.






M/M/1 and M/M/C simulation types for the queue component lambda and the number of entities

should be entered as inputs; for M/M/1/L and M/M/C/L type simulations, the limit also should

be provided for the queue component. For all simulation types, the server component mue

value should be entered for each server individually.






After running the simulation, the generated graph can be downloaded in png format.
